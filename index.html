Changing size

The width and height can be changed using the two dropdown boxes in the options section. Currently the minimum size is {{ page.minSize }} and the maximum size is {{ page.maxSize }}. The default size is {{ page.defaultSize }}. Size will not automatically update, you will have to click generate yourself for an update.

Handling Words

In this mode, words cannot be manually entered to be part of the game. However they are seleced at random from a list. The maximum number of words to be concidered to be part of the game is set in options. Currently the defualt max words is {{ page.maxWordsDefault }} and the highest value of the option is {{ page.maxWordsMax}}
